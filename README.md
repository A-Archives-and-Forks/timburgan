**How this works**

When you click a link, it opens a GitHub Issue with the required pre-populated text. Just push "Create New Issue". That will trigger a [GitHub Actions](https://docs.github.com/en/actions) workflow that'll update my profile with the new state of the board.

**Notice a problem?**

Raise an [issue](https://github.com/timburgan/timburgan/issues), and include the text _cc @timburgan_.
